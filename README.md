## 📌 About VibeChat

**VibeChat** is a modern and responsive chat interface web application built using **Next.js 16**, **React 19**, and **Tailwind CSS 4**.  
It is designed to provide a clean, smooth, and user-friendly chatting experience with an attractive UI, organized layout, and reusable components.

This project was developed as part of a company selection process to demonstrate frontend development skills, UI structuring, responsive design, and clean project organization.

---

## ✨ Features

- Modern chat application UI
- Responsive design for desktop and mobile
- Clean authentication screen structure
- Organized chat, pricing, support, and feature pages
- Reusable component-based architecture
- Custom global styling using Tailwind CSS
- Icons integrated using Lucide React
- Static assets managed inside the `public` folder
- Optimized Next.js app directory structure
